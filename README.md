start.ffhh
==========

Development
-----------

	aptitude install jekyll
	git clone git@github.com:freifunkhamburg/start-ffhh.git
	cd start-ffhh/
	jekyll serve -w

Deployment
----------

	aptitude install jekyll
	git clone https://github.com/freifunkhamburg/start-ffhh.git
	cd start-ffhh
	jekyll build
